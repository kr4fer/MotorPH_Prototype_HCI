<h2>Project Description</h2>
<ul>
  <li>This project is an <b>interactive dashboard prototype</b> designed for the MotorPH Payroll System.</li>
  <li>It focuses on transforming manual payroll workflows into a streamlined, user-centric digital interface.</li>
  <li>The design addresses key <b>HCI principles</b> such as visibility of system status, user control, and error prevention.</li>
</ul>

<h2>Core Objectives</h2>
<ul>
  <li><b>Automated Computation:</b> Eliminate manual errors in calculating Gross Pay, Net Pay, and government deductions.</li>
  <li><b>Streamlined Navigation:</b> Provide HR administrators with a centralized hub for employee and salary management.</li>
  <li><b>Data Integrity:</b> Organize employee profiles, tax identifications (SSS, PhilHealth, Pag-IBIG), and attendance records.</li>
</ul>

<h2>Key Features</h2>
<ul>
  <li><b>Secure Login Portal:</b> Role-based access for HR and Administrative staff.</li>
  <li><b>Interactive Dashboard:</b> Real-time overview of workforce statistics and payroll status.</li>
  <li><b>Employee Management:</b> Searchable directory for viewing and updating employee records.</li>
  <li><b>Payroll Processing:</b> Automated modules for calculating monthly salary cycles and statutory contributions.</li>
  <li><b>Attendance Tracking:</b> Visual logs for monitoring employee time-ins and leave requests.</li>
</ul>
